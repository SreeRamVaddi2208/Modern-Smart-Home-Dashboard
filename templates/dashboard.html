Data Preparation Steps Explained

1. Handling Missing Values

We filled missing numeric values with the median (middle value) of that column. This preserves the data distribution better than using the mean.

2. Removing Duplicates

Duplicate records were identified and removed. These are exact copies that can bias our analysis and waste computational resources.

3. Timestamp Conversion

Timestamps were converted from strings to proper datetime format, allowing us to extract time-based features like hour and day of week.

4. Feature Extraction

We created new useful features:

  • hour: Hour of day (0-23)
  • day_of_week: Day number (0=Monday, 6=Sunday)
  • is_peak_hour: Boolean indicating high energy usage hours
